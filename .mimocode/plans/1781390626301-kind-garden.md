## Mudanças

### `src/components/AiBrandsTicker.tsx`

1. **Substituir o array `brands`**: usar `{ name, src }` com URLs reais do CDN ao invés de unicode.
   - Manter as 9 marcas que possuem ícone no CDN
   - Remover Cohere e Stability AI (sem ícone disponível)

2. **Remover texto**: exibir apenas os ícones SVG (sem o nome da marca).

3. **Usar `<img>` tags**: renderizar cada ícone como `<img src={url} alt={name} width={28} height={28} />` com estilo `filter: grayscale(1) brightness(0.5) opacity(0.5)` para manter o visual sutil.

4. **Manter a animação JS existente**: o `requestAnimationFrame` + `translateX` funciona bem e não precisa de alteração.

### Detalhes de Implementação

```tsx
const CDN = "https://cdn.jsdelivr.net/gh/homarr-labs/dashboard-icons/svg";

const brands = [
  { name: "OpenAI", src: `${CDN}/openai.svg` },
  { name: "Anthropic", src: `${CDN}/anthropic.svg` },
  { name: "Google Gemini", src: `${CDN}/google-gemini.svg` },
  { name: "Meta", src: `${CDN}/meta.svg` },
  { name: "Mistral AI", src: `${CDN}/mistral-ai.svg` },
  { name: "Hugging Face", src: `${CDN}/hugging-face.svg` },
  { name: "DeepSeek", src: `${CDN}/deepseek.svg` },
  { name: "xAI Grok", src: `${CDN}/grok.svg` },
  { name: "Perplexity", src: `${CDN}/perplexity.svg` },

```

Cada item no ticker será apenas:
```tsx
<img
  src={brand.src}
  alt={brand.name}
  width={28}
  height={28}
  style={{ filter: "grayscale(1) brightness(0.5) opacity(0.5)", flexShrink: 0 }}
/>
```

Sem texto. Sem `<span>` com nome. Apenas a imagem.
