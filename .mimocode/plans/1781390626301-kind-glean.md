# Plano: Ticker de Ícones de IA — OráculoAI Site

## Conceito

Barra horizontal com logos das principais IAs do mundo, rolando continuamente (marquee) abaixo da navbar. Estilo "tech showcase" — ícones monochromáticos com brilho sutil no hover, fundo semi-transparente que combina com o dark theme do site.

1. **OpenAI** (ChatGPT) — logo flor/estrela
2. **Anthropic** (Claude) — logo A estilizado
3. **Google** (Gemini) — estrela 4 pontas
4. **Meta** (Llama) — logo infinito
5. **Mistral AI** — chama/ondulação
6. **Hugging Face** — emoji face
7. **xAI** (Grok) — X estilizado
8. **DeepSeek** — logo circular
9. **Cohere** — logo C
10. **Stability AI** — logo diamante

## Design do Componente

```
┌──────────────────────────────────────────────────────────────┐
│ [OpenAI] [Anthropic] [Google] [Meta] [Mistral] [HF] [xAI]  │  ← rolando →
│ [DeepSeek] [Cohere] [Stability] [OpenAI] [Anthropic] ...   │
└──────────────────────────────────────────────────────────────┘
```

- **Altura:** ~40px (py-2.5)
- **Fundo:** `bg-bg/60 backdrop-blur-sm` com `border-b border-border`
- **Ícones:** SVG inline, monochromáticos (fill `currentColor`), 20-24px
- **Cor dos ícones:** `text-muted` → hover `text-secondary`
- **Animação:** CSS `@keyframes marquee` com `transform: translateX()`, duração ~30s, linear, infinite
- **Pausa no hover:** `animation-play-state: paused` no container
- **Responsivo:** Em mobile, ícones menores (16px), velocidade igual
- **Acessibilidade:** `aria-label` no container, `prefers-reduced-motion` desabilita animação

## CSS Animation

```css
@keyframes marquee {
  0% { transform: translateX(0); }
  100% { transform: translateX(-50%); }
}

.ai-ticker-track {
  animation: marquee 30s linear infinite;
}

.ai-ticker:hover .ai-ticker-track {
  animation-play-state: paused;
}

@media (prefers-reduced-motion: reduce) {
  .ai-ticker-track {
    animation: none;
  }
}
```
